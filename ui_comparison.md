## タブ別の内容

### 🏢 建物本体タブ

```
■ 建物情報
  - 建築年月
  - 構造（RC造、重量鉄骨造、軽量鉄骨造、木造）
  - 階数
  - 延床面積（㎡）
  - 部屋数

■ 減価償却情報
  - 取得価額（円）
  - 取得年月日
  - 耐用年数（年）※構造により自動計算
  - 償却方法（定額法、定率法）
  - 残存価額（円）
  - 備考
```

### 🔧 建物付属設備タブ

```
ℹ️ 電気設備、給排水設備、冷暖房設備、昇降機設備などの
   建物付属設備を登録します。法定耐用年数は15年です。

■ 減価償却情報
  - 取得価額（円）
  - 取得年月日
  - 耐用年数（年）※デフォルト15年
  - 償却方法（定額法、定率法）
  - 残存価額（円）
  - 備考
```

### 🛣️ 構築物タブ

```
ℹ️ アスファルト舗装、コンクリート舗装、門・塀、フェンスなどの
   構築物を登録します。法定耐用年数は種類により10〜20年です。
   
   📝 アスファルト舗装: 10年
      コンクリート舗装: 15年
      門・塀: 20年

■ 減価償却情報
  - 取得価額（円）
  - 取得年月日
  - 耐用年数（年）※デフォルト15年
  - 償却方法（定額法、定率法）
  - 残存価額（円）
  - 備考
```

## ユーザーエクスペリエンスの向上

### シナリオ1: 建物付属設備の情報を入力する

**修正前:**
1. 「建物本体」タブで物件名を確認
2. 「建物付属設備」タブに切り替え
3. 物件名が見えなくなる ❌
4. 物件名を確認するために「建物本体」タブに戻る必要がある

**修正後:**
1. 基本情報で物件名を確認
2. 「建物付属設備」タブに切り替え
3. 物件名が常に表示されている ✅
4. そのまま入力作業を継続できる

### シナリオ2: 複数の物件を編集する

**修正前:**
- どの物件を編集しているか確認するために、毎回「建物本体」タブに戻る必要がある ❌

**修正後:**
- 基本情報が常に表示されているため、どの物件を編集しているか一目で分かる ✅

## 技術的な実装

### Bootstrap 5のコンポーネント

1. **カードコンポーネント**: 基本情報を視覚的にグループ化
2. **タブコンポーネント**: 複雑な情報を整理
3. **グリッドシステム**: レスポンシブな2カラムレイアウト
4. **フォームコントロール**: 統一されたUI

### カラースキーム

- **基本情報ヘッダー**: `bg-primary text-white`（青色）
- **情報アラート**: `alert-info`（水色）
- **必須項目**: `text-danger`（赤色）

### アイコン（Font Awesome 6.4.0）

- 📋 基本情報: `fa-info-circle`
- 🏢 建物本体: `fa-building`
- 🔧 建物付属設備: `fa-tools`
- 🛣️ 構築物: `fa-road`

## まとめ

基本情報をタブの外に配置することで、以下の改善を実現しました：

✅ **情報アクセスの向上**: 常に基本情報が表示される
✅ **作業効率の改善**: タブ切り替えの回数が減少
✅ **視覚的な明確さ**: 情報の階層構造が明確
✅ **ユーザビリティの向上**: 直感的で使いやすいUI

この変更により、物件管理アプリの使いやすさが大幅に向上しました。
